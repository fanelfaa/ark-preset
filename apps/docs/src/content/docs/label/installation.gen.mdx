## Installation

### CLI

Run the following command to add the component to your project:

```bash
npx @ark-preset/cli@latest add label
```

### Manual

Create the recipe file at `src/components/recipes/label.ts`:

```ts
import { tv, type VariantProps } from "tailwind-variants";

export const labelVariants = tv({
  base: "text-sm font-medium text-foreground leading-none peer-disabled:cursor-not-allowed peer-disabled:opacity-70",
  variants: {
    error: {
      true: "text-destructive",
    },
  },
});

export type LabelVariants = VariantProps<typeof labelVariants>;
```

> **Note:** Make sure your project has the Tailwind CSS theme variables set up (`--background`, `--foreground`, `--ring`, `--border`, etc.) or override the utility classes to match your design system.
