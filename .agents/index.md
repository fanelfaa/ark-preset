# Agent Rules Index

## Package-to-Rules Mapping

| Path               | Rules                         |
| ------------------ | ----------------------------- |
| `packages/*`       | `.agents/rules/moon/**/*.md`  |
| `apps/*`           | `.agents/rules/moon/**/*.md`  |
| `packages/solid/*` | `.agents/rules/solid/**/*.md` |
| `apps/docs/*`      | `.agents/rules/solid/**/*.md` |

## How to add rules

1. Create `.md` file in `.agents/rules/<domain>/`
2. Add entry to this index
3. Rules auto-load when working on matching paths
